Automated Email Sequence

6 Emails. Fully Automated.

All six emails are stored here and ready to import into Mailchimp. Each email is triggered automatically by a participant action — registration, Trove submission, or a scheduled date. Click Preview to see exactly what participants will receive.

01

Triggered: On Registration

Welcome — Investigative Briefing

Sent immediately after a team registers. Covers how the game works, K-Pop idol roles (Singer, Rapper, Dancer, Visual, Leader), K-Pop prep resource links, game start date, and links to the Suspects and Missions pages.

Trove 03 Complete — All Evidence Collected

Mailchimp Setup — How to Activate

  1. Create a free Mailchimp account at mailchimp.com
  2. Go to Campaigns → Email Templates → Create Template → Paste HTML
  3. Copy the HTML source from each Preview above and paste it into Mailchimp's HTML editor
  4. Set the From Name to Anna Im, Lead Detective and the From Email to your game email address
  5. In Automations, create a Customer Journey for each trigger (registration, form submission, date-based)
  6. In Phase 3, the TopKpop.io portal will connect to Mailchimp via API and trigger all emails automatically
